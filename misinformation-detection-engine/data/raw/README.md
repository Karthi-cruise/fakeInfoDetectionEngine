# Raw data

Place your raw fake news dataset here as **`fake_news.csv`**.

Expected columns (one of each group):
- Text: `text`, `content`, or `article`
- Title (optional): `title` or `headline`
- Label: `label`, `class`, or `target` (values like 0/1 or real/fake)

Download e.g. from Kaggle: "Fake News" dataset.
